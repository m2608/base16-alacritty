# base16-alacritty

[Base16](http://chriskempson.com/projects/base16/) themes template for [alacritty](https://github.com/alacritty/alacritty).

You could also try [this](https://github.com/aarowill/base16-alacritty) this template. For my taste colors there are a little bit off.

## How to use

- Generate a set of color themes. Probably you could use any base16 theme generator, I've only tested [mine](https://gist.github.com/m2608/48185612f371a7a0803ad1c329e59933).
- Add desired theme to your alacritty config. Or use simple script like [this](https://github.com/m2608/Stow/blob/master/stowed/.local/bin/alacritty-theme.sh) to switch theme.
